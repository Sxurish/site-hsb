# HSB Website (Next.js + Tailwind)

Corporate, conversion-focused marketing website for **HSB** in São Paulo, Brazil.

## Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion (subtle transitions)
- lucide-react
- TypeScript

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Validate

```bash
npm run lint
npm run build
```
